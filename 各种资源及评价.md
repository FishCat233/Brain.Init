这里会集合各种学习资源并附赠评价。

一些提示如下：

- 善用 `ctrl + f` 搜索以及 Github 的 Outline 大纲功能（右上角）
- 你可以把目录复制下来当 todo list (如果你想看完的话)
- 在看资源评价的时候注意考虑一下资源的推荐者，因为不同推荐者水平不同，适合他的不一定是适合你的

## 增加评价&资源注意事项

- 可以在资源后面附加评价，但是不要随意修改他人评价，否则 PR 会被退回
- 增加资源需要同时进行评价，第一个推荐者的评价放最前面
